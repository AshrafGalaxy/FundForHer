# **App Name**: ScholarPath

## Core Features:

- Profile Creation: User-friendly profile setup with educational background, financial status, and interests.
- Scholarship Discovery: Scholarship listings can be filtered and sorted by eligibility, level, field of study, location, and deadline.
- Bookmark Scholarships: Users can save scholarships of interest for future applications.
- AI Essay tool: AI-powered essay feedback tool: users receive intelligent suggestions on the essays they write as part of their applications

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a sense of calm and support.
- Background color: Very light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Light peach (#FFDAB9) for highlighting important actions and elements.
- Font pairing: 'Poppins' (sans-serif) for headings and 'PT Sans' (sans-serif) for body text, providing a modern and readable design.
- Use of clean and modern icons related to education, finance, and personal growth.
- Mobile-first, responsive layout ensuring accessibility and usability on various devices.
- Subtle animations for a smoother user experience, such as loading animations or feedback animations on button presses.
